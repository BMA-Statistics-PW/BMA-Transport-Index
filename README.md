# 🚦 BMA Transport Index Dashboard

A data-driven dashboard framework for the BMA Transport Index, integrating multi-source transport data into actionable insights.
Features include KPI monitoring, trend analysis, modal share visualization, and transport performance metrics to support urban mobility management and strategic planning.

---

## 📌 Overview

BMA Transport Index Dashboard เป็นระบบแสดงผลข้อมูลด้านการขนส่งของกรุงเทพมหานครในรูปแบบ Interactive Dashboard
พัฒนาเพื่อสนับสนุนการวิเคราะห์เชิงนโยบาย การติดตามสถานการณ์ และการวางแผนระบบขนส่งในเมือง

ระบบนี้รวมข้อมูลจากหลายแหล่ง (Multi-source Data) เพื่อแปลงเป็นข้อมูลเชิงลึก (Insights) ที่สามารถนำไปใช้ในการตัดสินใจได้จริง

---

## 🎯 Objectives

* ติดตามตัวชี้วัดด้านการขนส่ง (Transport KPIs)
* วิเคราะห์แนวโน้มผู้โดยสาร (Passenger Trend Analysis)
* แสดงสัดส่วนรูปแบบการเดินทาง (Modal Share)
* ประเมินประสิทธิภาพระบบขนส่ง (Transport Performance)
* สนับสนุนการวางแผนเชิงนโยบาย (Policy Planning)

---

## 📊 Key Features

### 1. KPI Dashboard

* ผู้โดยสารรถไฟฟ้า (เฉลี่ย/วัน)
* ผู้โดยสารรถโดยสาร
* ผู้โดยสารเรือ
* ความเร็วเฉลี่ยการจราจร

### 2. Trend Analysis

* แนวโน้มผู้โดยสารรายเดือน
* เปรียบเทียบแต่ละระบบขนส่ง
* รองรับการวิเคราะห์เชิงเวลา (Time Series)

### 3. Modal Share

* แสดงสัดส่วนการเดินทางของแต่ละระบบ
* รองรับการวิเคราะห์เชิงโครงสร้าง (Structure Analysis)

### 4. Transport Performance

* ความตรงต่อเวลา
* ความพึงพอใจ
* ความครอบคลุมพื้นที่
* ประสิทธิภาพระบบ

### 5. Data Table

* ตารางสรุปข้อมูลเชิงสถิติ
* รองรับการตรวจสอบรายละเอียดเชิงลึก
