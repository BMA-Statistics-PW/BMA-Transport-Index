เกี่ยวกับโครงการ

BMA Urban Transport Index จัดทำโดยกลุ่มงานสถิติและวิจัย กองนโยบายและแผนงาน สำนักการจราจรและขนส่ง กรุงเทพมหานคร

📋 แหล่งข้อมูลหลัก

  • สำนักการจราจรและขนส่ง (สจส.)
  • องค์การขนส่งมวลชนกรุงเทพ (ขสมก.)
  • การรถไฟฟ้าขนส่งมวลชนแห่งประเทศไทย (รฟม.)
  • การรถไฟแห่งประเทศไทย (รฟท.) / รฟฟท.
  • สำนักงานนโยบายและแผนการขนส่ง (สนข.) — BTDM
  • TomTom Traffic Index 2025

🛠️ เทคโนโลยี

  • HTML5 · CSS3 (CSS Custom Properties)
  • JavaScript (Vanilla ES2020)
  • Chart.js 4.4.1
  • Google Fonts: Sarabun + Chakra Petch
  • GitHub Pages + GitHub Actions CI/CD
  • Data: CSV + JSON (DCAT standard)

⚠️ ข้อควรทราบ

  • ขสมก. ใช้ปีงบประมาณ (ต.ค.–ก.ย.)
  • ระบบรางใช้ปีปฏิทิน (ม.ค.–ธ.ค.)
  • ปี 2563–2564 ได้รับผลกระทบจาก COVID-19
  • ข้อมูลความเร็วเก็บจากสำรวจภาคสนาม ไม่ใช่ GPS
  • TomTom ใช้เพื่อ benchmark เท่านั้น
